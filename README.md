# Zhuangzu

#### 介绍
本项目是基于原生Andorid开发的简单的壮族信息浏览app，后端使用bmob来进行快速开发以及网上存储，采用了recycleview组件进行翻页浏览，内置了SildeMenulibrary-release来实现主页面和用户界面之间滑动切换。
